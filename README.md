# Sistema de Combate por Turnos (Python)

Este proyecto es un juego en consola desarrollado en Python que simula un sistema de combate 1 vs 1 por turnos, en base a la lógica de los juegos clásicos de Pokémon.

## Características

- Sistema de combate por turnos
- Sistema de puntos de vida (HP)
- Probabilidad de esquivar ataques
- Sistema de golpes críticos
- Sistema de curación con pociones
- IA básica que toma decisiones aleatorias

## Tecnologías utilizadas

- Python 3
- random
- time

## Cómo ejecutarlo

1. Clonar el repositorio o descargar el archivo.
2. Ejecutar en la terminal:
python Sistema_de_combate_por_turnos.py


## Lo que demuestra este proyecto

- Uso de bucles `while`
- Condicionales complejos (`if/elif`)
- Lógica de estados
- Generación de números aleatorios
- Diseño básico de IA
